# Scientific Calculator Application

This project is a scientific calculator application built with Python. It provides various mathematical operations and functionalities, including basic arithmetic, trigonometric functions, and square root calculations.

## Project Structure

```
backend
├── src
│   ├── __init__.py
│   ├── calculator.py
│   ├── models
│   │   └── math_models.py
│   ├── repositories
│   │   └── math_repositories.py
│   ├── services
│   │   └── calculator_services.py
│   └── utils
│       └── helper_functions.py
├── config
│   └── settings.json
├── requirements.txt
├── run.py
└── README.md
```

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the backend directory:
   ```
   cd backend
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

To run the application, execute the following command:
```
python run.py
```

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division
- Trigonometric functions: sine, cosine, tangent
- Square root calculation
- Data storage and retrieval for operations
